# 悟空库
悟空库全新五合一版本（犁牛、松鼠、谷歌、Cydia/Sileo源、防火墙集成于一体）

全新架构，源码编译，无漏洞无后门。 

数据库凭证加密、连接加密、程序加密防盗库攻击

支持本地部署、网络部署、本地+网络混合部署，防止DDOS攻击导致无法作业。

独家插件管理：系统集成Cydia/Sileo源，有根无根互转，逆向改IP。   

   
功能如下：

- 主页

- 游戏管理
  - 游戏类型
  - 游戏档位

- 库存管理
  - 游戏库存
  - 库存预警
  - 档位库存
  - 账户库存
  - 入库记录
  - 出库记录
  - 入库统计
  - 出库统计
  - 库存记录

- 账户管理
  - 子账户管理
  - 谷歌验证器
  - 设备
  - 用户Token
   
- 插件管理
  - 插件（上传/下载/转有根/转无根/改IP）
  - 插件源设置
  - 插件分类
    
参数
防火墙
