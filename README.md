# 悟空库
犁牛、松鼠、谷歌库三合一，完全兼容犁牛库、松鼠库、谷歌库插件。   

系统进行连接加密、程序加密、数据加密，多重保障凭证安全，谨防盗库。

支持本地部署、网络部署、本地+网络混合部署，防止DDOS攻击导致无法作业。

   
功能如下：

- 主页

- 游戏管理
  - 游戏类型
  - 游戏档位

- 库存管理
  - 游戏库存
  - 档位库存
  - 账户库存
  - 入库记录
  - 出库记录
  - 入库统计
  - 出库统计
  - 库存记录

- 账户管理
  - 子账户管理
  - 谷歌验证器
  - 设备
  - 用户Token
参数
